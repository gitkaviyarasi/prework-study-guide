HTML

  • The head element contains information about the webpage.
  • The Body element represents the visible content shown to the user.
  • The DOCTYPE is a declaration.
  • Title represents the title on the browser.
  • list - li stands for list
  • ul - unordered list and ol stands for ordered list.

CSS

  • CSS gives the style for the web page.
  • Magin indicates how much space we want around the outside of an element.
  • Padding indicates how much space we want the content inside an element.
  • We can give color in hexadecimal format, names, rgb,hsl
  • There are element selector,id selector, and class selector.

GIT

  • git status:Checks what branch we are currently .
  • git checkout -b branch-name : creates a new branch and switches to it.
  • git add -A: add all changes to the current branch.
  • git commit -m "comments" : commits

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
